# INU LAB — Venison 03 product page

A single-page product showcase for INU LAB Venison 03, a single-protein raw frozen recipe for dogs.

The layout follows a progressive-disclosure structure: hero, outcome numbers, benefits organised by outcome, the ColdLock raw process, a benefit-led nutrient analysis, trust markers, and a trade stockist call to action.

## Brand

- Warm White base, Ink Navy text, ochre accent, red meat family colour for this recipe.
- DM Sans headings, Inter body.
- Light and dark mode with a manual toggle.

## Structure

```
index.html   markup
style.css    brand tokens and layout
app.js       theme toggle and scroll reveal
assets/      seal mark, hero image, food image
```

## Run

Open `index.html` in a browser. No build step.

Nutrient figures are drawn from the INU LAB brand kit: Venison 03 is 16 / 12 / 66 / 2.5 / 3 (protein / fat / moisture / fibre / ash) per 100 g, 500 g tray.
